## 4. 执行命令  
> * 生成序列文件,将文字编码为数字,不足补零  
`python data_process.py`  
> * 生成word2vec向量,包括编码向量和解码向量  
`python word2vec.py`  
> * 训练网络  
`python train.py`  
> * 测试  
`python test.py`  
> * 模型评分  
`python score.py`  
> * 智能问答  
`python chat_robot.py`  
> * 绘制word2vec向量分布图  
`python word2vec_plot.py`
